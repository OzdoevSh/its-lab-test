# ITS_Lab Test Task

## **Getting Started**

### **Требования**
- Node.js (v16 или выше)

### **Установка**

1. Клонировать репозиторий
2. Установить зависимости:
  ```
  npm install
  ```

### **Доступные скрипты**

1. Запуск проекта:
  ```
  npm run dev
  ```
2. Запуск ESLint:
  ```
  npm run lint
  ```
2. Тестирование:
  ```
  npm run test
  ```
